# Generate SSH Key Pair Demo

This example demonstrates the fundamental process of generating an SSH key pair (private and public keys) using the `ssh-keygen` command. It creates a temporary directory, generates the keys, explains their purpose, and then cleans up the generated files. This illustrates the core concept of asymmetric cryptography central to SSH security.

## Language

`bash`

## How to Run

1. Save the code as `generate_ssh_keys.sh`.
2. Make the script executable: `chmod +x generate_ssh_keys.sh`.
3. Run the script: `./generate_ssh_keys.sh`.
